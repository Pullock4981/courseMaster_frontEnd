## How to Use These Questions

1. Go to **Admin Dashboard** → **Manage Courses** → **Create Course** or **Edit Course**
2. Add a **Module** (e.g., "HTML Basics")
3. Add a **Lesson** (e.g., "Introduction to HTML")
4. Scroll down to **"Quiz Questions (Optional)"**
5. Click **"+ Add Question"**
6. Enter the question text
7. Fill in all 4 options
8. Click the **radio button** next to the correct answer (this sets the `correctIndex`)
9. Repeat for more questions

## Notes

- **Correct Index** starts at 0 (first option = 0, second = 1, third = 2, fourth = 3)
- Always provide exactly 4 options for each question
- Make sure to select the correct answer by clicking the radio button
- You can add multiple questions per lesson
- Questions are optional - lessons can have videos and assignments without quizzes
